# Phonetact

Веб-приложение для упрощения работы с контактами и телефонными номерами.

## Описание

Phonetact — это инструмент, который позволяет быстро создавать виртуальные визитные карточки и делиться контактной информацией через QR-коды или прямые ссылки. Приложение также предоставляет возможность связаться с номером телефона через различные мессенджеры и сервисы без необходимости сохранять контакт в телефонной книге.

## Основные возможности

- **Создание виртуальных визитных карточек** — генерация контактных карточек с полной информацией (имя, телефон, email, адрес и др.)
- **Генерация QR-кодов** — создание QR-кодов в формате vCard для быстрого обмена контактами
- **Быстрая связь по номеру** — возможность позвонить или написать в мессенджеры (WhatsApp, Telegram, Viber) без сохранения номера в контакты
- **Форматирование телефонных номеров** — автоматическое приведение номеров к единому формату

## Технологии

- **Backend**: Laravel 11
- **PHP**: 8.2+
- **QR-коды**: Endroid QR Code

## Установка

1. Клонируйте репозиторий
2. Установите зависимости: `composer install`
3. Настройте файл `.env`
4. Запустите миграции: `php artisan migrate`
5. Запустите сервер: `php artisan serve`
